# Sistema de Refeições

App mobile em React Native para registrar, editar, excluir e acompanhar refeições diárias.

## Funcionalidades

- Cadastro de refeições com nome, descrição, data, horário e marcação de dentro/fora da dieta
- Edição de refeições existentes
- Exclusão com confirmação
- Lista organizada por data
- Estatísticas de progresso da dieta
- Tela de feedback após cadastro
- Dados persistidos localmente com AsyncStorage

## Tecnologias

- Expo
- React Native
- React Navigation (Native Stack)
- AsyncStorage
- styled-components
- TypeScript

## Como executar

1. Instale as dependências:

```bash
npm install
```

2. Inicie o projeto:

```bash
npm start
```

3. Abra no Expo Go ou no emulador desejado.
